# 02 - Game Design

## Hlavní postava
- Jméno: Hráč
- Vlastnosti: Skoky, rychlost pohybu

## Herní svět
- Vertikální levely s platformami různých velikostí
- Některé platformy pohyblivé
- Překážky a hazardy (např. trny, díry)
- Jednoduché vizuální prostředí (pixel art / 2D styl)

## Ovládání
- Šipky / WASD pro pohyb
- Mezera / Space pro skok
- Držení klávesy pro delší skok

## Zvuky a hudba
- Menu a lobby
- Skoky a dopady
- Checkpoint
- Game over

## Level design
- Každá úroveň bude težší a těžší
